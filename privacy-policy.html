Privacy Policy

Last Updated: October 3, 2025

Summary: This oscilloscope simulator runs entirely in your web browser. We do not collect, store, or transmit any of your data. Your privacy is completely protected.

1. Information We Collect

We collect NO personal information. This application is a client-side web application that runs entirely in your browser. All functionality operates locally on your device.

2. Data Storage

All data generated by the oscilloscope simulator, including:

  • Signal configurations
  • Measurement data
  • Control settings
  • Display parameters

...is stored temporarily in your browser's memory only and is automatically deleted when you close the application or refresh the page.

3. Cookies and Tracking

We do NOT use:

  • Cookies
  • Analytics tools (Google Analytics, etc.)
  • Tracking pixels
  • Third-party scripts
  • Advertising networks

4. Third-Party Services

This application does not connect to any external servers or third-party services. It operates completely offline once loaded.

5. Local Storage

Currently, this application does not use browser local storage. If future versions implement settings persistence, you will be notified and given the option to opt-in.

6. Data Sharing

Since we don't collect any data, we have nothing to share with third parties. Your usage of this application is completely private.

7. Children's Privacy

This application is safe for users of all ages. We do not collect information from anyone, including children under 13.

8. Your Rights

Because we don't collect any personal data:

  • There is no data to access
  • There is no data to delete
  • There is no data to export
  • There is no data to correct

You have complete control over this application simply by using it.

9. Security

While we don't collect data, we still prioritize security:

  • The application runs entirely in your browser's sandboxed environment
  • No data is transmitted over the internet
  • No external resources are loaded after initial page load

10. Browser Permissions

This application does not request any special browser permissions. It only uses standard web APIs available to all web pages.

11. Changes to This Privacy Policy

If we ever change how this application handles data, we will update this privacy policy and update the "Last Updated" date at the top of this page.

12. Open Source

This is an open-source educational tool. You can review the complete source code to verify that no data collection occurs.
